# speech-conversion
ステレオミキサーから音声を翻訳する

## 事前準備

### GCPアカウントを作る

### サービスアカウントを作る

### サービスアカウントのキーを作る

### Google Cloud Speech-to-Text APIを有効にする

https://console.developers.google.com/apis/api/speech.googleapis.com/overview?project=1065481360790

## 実行方法

### キーを環境変数にセットする

export GOOGLE_APPLICATION_CREDENTIALS="【JSONファイル名】"

### プログラムを起動する

node index.js

### プログラムを終了する

Ctrl+C
